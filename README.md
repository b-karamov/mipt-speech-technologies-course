## Описание репозитория
Материалы по практическим заданиям в рамках курса "Речевые технологии"

### Структура репозитория

```text
.
├── audio/                         # Исходные записи и сохранённые noisy/denoised WAV
├── mel_spectrogram/               # Модуль mel_spectrogram.py и ноутбук с примером вычисления mel-спектрограммы
├── noise_cancellation/            # Рабочая директория с ноутбуками по шумоподавлению
│   ├── spectral_subtraction.ipynb # Пайплайн спектрального вычитания (STFT + метрики)
│   └── wiener_filter.ipynb        # Аналогичный пайплайн, но с винеровской фильтрацией
└── requirements.txt               # Зависимости
```
